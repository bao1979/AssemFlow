# AssemFlow Tutorial · Learning Map

> A gradual introduction to Assembly Flow Programming (AFP), from first intuition to a real project.
> The Chinese version remains the source of truth; this English set is a derived translation view.

## How To Use This Tutorial

The tutorial is split into five parts. You can stop after any part and still walk away with something complete:

- Just want to know what AFP is? Start with Part One. It takes about five minutes.
- Want to try it by hand? Go through Part Two.
- Want to judge the discipline seriously? Read Part Three.
- Want the theory and research framing? Continue with Part Four.
- Want to bridge teaching examples to a real project? Finish with Part Five.

## Teaching Style

- **Intuition before terminology**: first make it feel concrete, then give it a name.
- **Contrast-driven**: key points are paired with counterexamples showing what happens without AFP.
- **Runnable**: every lesson can be tested by actually running something.
- **Code as textbook**: the tutorial explains why; the details live in the real code inside `experiments/`.
