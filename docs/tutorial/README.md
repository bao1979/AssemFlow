# AssemFlow 教程 · 学习地图

> 装配流编程（AFP）渐进式教程。由浅入深，任何人都能跟着走。
> 中文为内容真相源，英文版按 `language.md` 双语规范派生。

## 怎么用这份教程

教程分四部分，**可中途下车**——在任何一部分停下都有完整收获：

- 只想知道这是什么 → 第一部分（5 分钟）
- 想动手试 → 第二部分（跟着拼）
- 想认真评估这范式 → 第三部分
- 想深究理论 / 参与贡献 → 第四部分

## 教学法约定

- **直觉先于术语**：先给感觉，再给名字。
- **对比驱动**：关键处配"不用 AFP 会怎样"的反例。
- **可运行**：每课能亲手跑、看到输出。
- **代码即教材**：教程讲"为什么"，细节看 `experiments/` 里带教学注释的真实代码。
